# Codea Desarrollos

This is a Next.js project for Codea Desarrollos, a software development company.
The design is inspired by Klime Studio, featuring a dark theme, large typography, and an image gallery.

## Getting Started

First, run the development server:

\`\`\`bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
\`\`\`

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Features

- **Header**: Navigation links and logo.
- **Hero**: Large introductory text.
- **Gallery**: Horizontal scrolling image gallery.
- **Footer**: Call to action.

## Tech Stack

- Next.js 15
- Tailwind CSS
- React
